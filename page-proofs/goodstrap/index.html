Getting started

Downdload

To start using Goodstrap, all you have to do is push on button download.

Download

The file structure

								
	goodstrap/
	├── css/
	│   ├── breadcrumb.css
	│   ├── buttons.css
	│   ├── forms.css
	│   ├── menu.css
	│   ├── pagination.css
	│   ├── reset.css
	│   └── style.css
	│     
	└── index.html /*base page*/

In the css/ folder includes the styles of each component presented in this framework. Use the index.html page for a quick dip in Goodstrap.

HTML5 doctype

Goodstrap makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your projects. 

									
<!DOCTYPE html>
<html>
...									
</html>

Include files

To work successfully with Goodstrap simply connect the style.css file which contains the reference to the rules of each component and reset styles.

Next style.css, you should connect the link to the icons and font

Basic template

Copy the HTML below to begin working with a minimal Goodstrap document.

								
<!DOCTYPE html>
<html>
    <head>
        <title>My first page with Goodstrap</title>
        <meta charset="utf-8" />
		
        <!--Icon and font-->
        <link href='https://fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css' />
        <link href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css" rel="stylesheet" />
        <!--Goodstrap-->
        <link href="css/style.css" rel="stylesheet" />

    </head>
    <body>
        <h1>Hello team internship</h1>

    </body>
</html>

Buttons

Default Buttons

To create a button, add the .btn classname to any <a> or <button> element.

Link 
								
<button class="btn">Button</button>

<a href="#" class="btn">Link</a>

Modifiers

To change the appearance of the button you want add class modifiers to an item that already has .btn classname.

Button Sizes

There are one button sizes modifiers .__lg . 

								
<button class="btn __lg">Large Button</button>

Button Colours

There are two button colour modifiers .__primary or .__danger. 

								
<button class="btn __lg __primary">Primary</button>

<button class="btn __lg __danger">Danger</button>

Disabled Button

To mark a button as disabled, add the attribute disabled. 

								
<button class="btn __lg __primary" disabled>Primary</button>

<button class="btn __lg __danger" disabled>Danger</button>

Forms

Default Forms

To create a basic input, add the .it(input) classname to <input> and .itx (textarea) classname to <textarea> element.

NOTE: the default size of the input, fills the entire width of the parent container.

								
<input class="it" type="text" placeholder="Default" />

<textarea class="itx" placeholder="Default"></textarea>

If you are aiming to create valid and accessibility html forms, be sure you are using the tag label. More 

								
<div class="field-it">
    <label for ="example1"></label>
    <input class="it" id="example1" type="text" placeholder="Normal" />
</div>

Input with icon

NOTE: icon must be placed within a .field-it (field-input). 

								
<div class="field-it">
    <label for ="example1"></label>
    <input class="it" id="example1" type="text" placeholder="Normal" />
    <i class="fa fa-search field-it_ic"></i>
</div>

Input with badge

NOTE: span with class .field-it-group_badge must be placed within a .field-it. 

								
<div class="field-it-group">
    <label for ="example1"></label>
    <input class="it" id="example1" type="text" placeholder="Normal" />
    <span class="field-it-group_badge">
        <i class="fa fa-search"></i>
    </span>
</div>

Modifiers

To change the appearance of the input you want add class modifiers to an item that already has .it (input) classname.

To change group forms, add class modifiers to the parent classes wrapper .field-it or .field-it-group.

Input Sizing

There are two input sizes modifiers .__sm or .__lg . 

								
<input class="it __sm" type="text" placeholder="Small size" />

<div class="field-it __lg">
    <label for="example1"></label>
    <input class="it" id="example1" type="text" placeholder="Normal" />
    <i class="fa fa-search field-it_ic"></i>
</div>

<div class="field-it-group __lg">
    <label for="example1"></label>
    <input class="it" id="example1" type="text" placeholder="Normal" />
    <span class="field-it-group_badge">
        <i class="fa fa-search"></i>
    </span>
</div>

Background Input

The class.__bg-dark changes the background of the input dark. 

								
<input class="it __sm __bg-dark" type="text" placeholder="Dark background" />

<div class="field-it __lg __bg-dark">

    <i class="fa fa-search field-it_ic"></i>

<textarea class="itx __bg-dark" placeholder="Dark background"></textarea>

<input class="it __sm __error" type="text" placeholder="Error" />
